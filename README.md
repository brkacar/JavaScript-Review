# JavaScript Review 📘

A personal reference and practice repository covering core JavaScript fundamentals.
Each topic is isolated in its own file or folder, making it easy to revisit specific
concepts quickly — ideal for refreshers and interview preparation.

---

## 📁 Repository Structure

```
JavaScript-Review/
│
├── ControlFlows-Conditionals_Loops/   # if/else, switch, for, while, for...of
├── OperatorsAndExpressions/           # Arithmetic, logical, comparison, assignment
├── VariablesAndDataTypes/
│   ├── dataTypes.js                   # Primitives, reference types, type checking
│   └── variables.js                   # var, let, const, scope, hoisting
│
├── arrays.js                          # Array methods and iteration
├── async.js                           # Promises, async/await, error handling
├── errors.js                          # try/catch, custom errors, finally
├── functions.js                       # Declarations, expressions, arrow functions
├── json.js                            # JSON.parse(), JSON.stringify()
├── objects.js                         # Object literals, methods, destructuring
└── strings.js                         # String methods and manipulation
```

## 🚀 Getting Started

No dependencies or setup required. Just clone and run any file with Node.js:

```bash
git clone https://github.com/brkacar/JavaScript-Review.git
cd JavaScript-Review
node arrays.js
```

## 🎯 Purpose

This repo serves as a hands-on study guide to consolidate JavaScript knowledge,
with small and focused code examples that are easy to read and run independently.
